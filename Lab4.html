¿Qué diferencias y semejanzas hay entre Java y JavaScript?

  • Javascript no es compilado, es interpretado.
  • Javascript está basado en objetos, Java está orientado a objetos, JavaScript no utiliza clases, herencias o técnicas habituales en la OOP.
  • en Javascript no existe la declaración de tipos de variables, se encarga de eso automaticamente.
