¿Qué es ODBC y para qué es útil?

Open Data Base Conectivity, Es una interfaz de Microsoft para recuperar registros de una base de datos en un entorno heterogéneo de sistemas relacionales y no relacionales de manejo de base de datos.

¿Qué es SQL Injection?

Es una técnica maliciosa mediante la cual un intruso puede insertar comandos de SQL en el código a través de un input en una página web, con el propósito de afectar la integridad de una base de datos.

¿Qué técnicas puedes utilizar para evitar ataques de SQL Injection?

Usando parámetros de SQL en los inputs.
