¿Qué es SQL injection y cómo se puede prevenir?

Es una técnica maliciosa mediante la cual un intruso puede insertar comandos de SQL en el código a través de un input en una página web, con el propósito de afectar la integridad de una base de datos.

¿Qué importancia tiene AJAX en el desarrollo de RIA's (Rich Internet Applications?

Las RIA’s son la combinación entre aplicaciones de escritorio y web. Mejoran la experiencia el usuario y la productividad de éste con interfaces más intuitivas. AJAX permite que las páginas sean más dinámicas e intuitivas permitiéndonos cubrir mejor las necesidades del cliente.

¿Qué implicaciones de seguridad tiene AJAX? ¿Dónde se deben hacer las validaciones de seguridad, del lado del cliente o del lado del servidor?

Se debe validar por ambos lados, del lado del servidor por seguridad y del lado del cliente para que no se pierda tiempo y se notifique cuanto antes cuando falta algún campo por llenar o cuando no se está llenando en el formato indicado.

¿Qué es JSON?

JavaScript Object Notation, es una sintaxis usada para intercambiar y almacenar datos. Es una alternativa de XML, pero es más fácil de usar. Usa la misma sintaxis que JavaScript pero es puro formato al igual que XML.
